# 注释
> 书写注释是一个非常好的习惯。

## 单行注释
```java
// 这是一个单行注释
```

## 多行注释
```java
/* 这是一个
   多行注释 */
```

## JavaDoc: 文档注释
```java
/**
 * 这是一个文档注释
 * @author 作者名
 */
```
